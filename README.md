# Stable Diffusion Tags

## 项目依赖

- [MongoDB >= 6.0.2](https://www.mongodb.com/)
- [NodeJS >= 16.15.1](https://nodejs.org/)

## 快速开始

```bash
npm run dev
# or
yarn dev
```

打开 [http://localhost:3200](http://localhost:3000) 查看页面

## 开发

### 数据

``` bash
cd ./services

yarn start:dev
```

打开 [http://localhost:3201/doc](http://localhost:3201/doc) 查看接口文档
