# Stable Diffusion Tags

## 项目依赖

- [MongoDB >= 6.0.2](https://www.mongodb.com/)
- [NodeJS >= 16.15.1](https://nodejs.org/)

## 快速开始

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
